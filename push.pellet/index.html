Pushpellet API

API Parameters

message

  • required
  • the message to send

phone

  • optional
  • the phone number to send the message to (Sprint)

email

  • optional
  • the email address to send the message to

Send to an e-mail address

cURL

curl "{ pusher URL }/?message=the message to send&email=email@example.com"

HTTP (GET)

{ pusher URL }/?message=the message to send&email=email@example.com

jQuery

$.get('{ pusher URL }/?message=the message to send&email=email@example.com', null, status => {
  (status === 'OK')?
    console.log('Message sent'):
  console.warn(status)
}, 'text');

JavaScript

fetch('{ pusher URL }/?message=the message to send&email=email@example.com')
  .then(response => response.text())
  .then(status => {
    (status === 'OK')?
      console.log('Message sent'):
    console.warn(status)
  })
