Paramour is a client-side, preprocessed JavaScript (JS) language, based off of CoffeeScript (CS), Java, Python, Swift, and new ECMAScript features. Though JavaScript is powerful alone, Paramour enhances and simplifies JavaScript's best and worst qualities.

It is assumed that you, the reader, know at least the basics of JavaScript, and if not, see "JavaScript" by W3Schools.

Paramour is a multi-paradigm language (17):

Paramour's Programming Paradigms

  • Block Structured
  • Conceptual
  • Dynamic
  • End User
  • Event Driven
  • Expression Oriented
  • Functional
  • Imperative
  • Modular
  • Object Oriented
  • Procedural
  • Protocol Oriented
  • Prototype Oriented
  • Reflective
  • Scripting
  • Structured

var numbers = [9, 8, 7, 6, 5, 4, 3, 2, 1], first3, middle3, last3;

# ... means splice
# [a..b] = [...] is also splice
# [a...b] = [...] is also splice
​
first3 = numbers[0..3];
  # numbers.SLICE(from 0, towards the end 3 indexes)
  # "numbers from[0, 2] => [9, 8, 7]"
​
middle3 = numbers[3..3];
  # numbers.SLICE(from 3, towards the end 3 indexes)
  # "numbers from [3, 5] => [6, 5, 4]"
​
last3 = numbers[-2..3];
  # numbers.SLICE(from the end 3 indexes, towards to the end 3 indexes)
  # "nubers from[6, 8] => [3, 2, 1]

numbers[0..5] = ["nine", "eight", "seven", "six", "five"];
  # numbers.SPLICE(0, 5, "nine", "eight", "seven", "six", "five")

"text/paramour" Script Tags

If you want to run a Paramour script that is loaded via a URL, then this section is for you.
To use this feature, you must have a <script type="text/paramour"> tag, or type="text/par" if you feel a bit lazy. You can download the latest version here, about 65.0 Kb.
Afterwards, you simply call Paramour.compile(true) and Paramour will handle the rest from there.
A simple example: 

<head>
  <script src="par.min.js"></script>
  <script src="Gumball.par" type="text/par"></script>
</head>
<body onload="Paramour.compile(true)">
  <!-- blah blah blah -->
</body>
