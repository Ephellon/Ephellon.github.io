Mi/o

Mi/o is an optimized LZW algorithm, that also allows the use of the Base64 algorithms (atob and btoa in JavaScript).

More about Mi/o's algorithms

Mi/o is called on as Mio.enc(String data[, Boolean hybrid-only]) to encode data, and Mio.dec(String data) to decode it.

  • Mio.enc is also available as Mio.Mi
  • Mio.dec is also available as Mio.Mo
  • Boolean hybrid-only tells Mio to leave the encoded data in LZW format true, or Base64 format false; default is false
